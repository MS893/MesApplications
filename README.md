# SkyDev Apps - Écosystème Android pour Pilotes VFR

![SkyDev Banner](assets/img/aeroclub.png) <!-- Utilisation de l'icône Aéroclub comme illustration par défaut -->

Ce projet est une vitrine web moderne présentant une suite d'applications Android conçues spécifiquement pour l'aviation générale. Développée avec **Bootstrap 5**, cette plateforme offre une expérience utilisateur fluide et esthétique pour découvrir les outils destinés aux pilotes VFR, instructeurs et examinateurs.

## 🚀 Les Applications

L'écosystème **SkyDev** se compose de quatre piliers essentiels :

| Application | Description | Statut |
| :--- | :--- | :--- |
| **Aéroclub** | Le couteau suisse du pilote et mémo VFR indispensable pour la gestion de l'activité. | ✅ Disponible |
| **Radio PPL** | Entraînement à la phraséologie radio pour maîtriser les communications VFR. | ⏳ Bientôt |
| **Perfos** | Calculateur de performances au décollage et à l'atterrissage pour une sécurité maximale. | ⏳ Bientôt |
| **FI-FE** | Assistant numérique dédié aux Flight Instructors (FI) et Examiners (FE). | ⏳ Bientôt |

## 🛠️ Technologies Utilisées

- **Frontend :** HTML5, Vanilla CSS3 (Custom animations & Glassmorphism).
- **Framework CSS :** [Bootstrap 5](https://getbootstrap.com/).
- **Typographie :** Google Fonts (Outfit pour les titres, Inter pour le corps).
- **Icônes :** Custom PNG assets.

## 💻 Installation Locale

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-username/MesApplications.git
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur ou utilisez un serveur local comme `live-server` :
   ```bash
   npx live-server
   ```
